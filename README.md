# mpum-2026-miniprojects
This repository contains 3 small projects made during the course "Probabilistic methods in computer science" course. The reports and comments in the code are written in Polish.

# Project 1: linear regression
Linear regression implementation using gradient methods and analytic solution. I compared the performance of models using different base functions(exponential, sigmoid, linear combination, polynomial) and regularization (lasso, ridge, elastic net).
# Project 2: logistical regression and naive bayes classifier
In this project I trained logistical regression and naive bayes classifier to detect cancer in patients. I fine-tuned their hyperparameters and compared the performance.
# Project 3: SVM, decision tree, KNN
In this project i trained linear (SVM) and non-linear (decision tree, KNN) clasifiers to detect phishing. For SVM i tested different hyperparams and kernel function - needed to optimize the training to make it possible. For non-linear models i tested different hyperparams and compared the performance of three models.
